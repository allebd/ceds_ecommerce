== Description ==

Keep Calm and E-Comm provides a sleek, modern and responsive theme layout that allows you to display your WooCommerce shop in a clear and attractive manner. This theme is designed, from the ground up, to integrate seamlessly with the <a href="https://wordpress.org/plugins/order-tracking/">Status Tracking</a> plugin. It's easy to set up and offers the perfect interface for your customers to place and track their orders.

* Modern responsive layout
* WooCommerce shop ready
* Status Tracking plugin integration
* Feature-rich slider that pulls slides either from a dedicated custom post type, straight from your product catalog or from WooCommerce
* Testimonials
* Multiple page layouts
* Blog layout
* Theme options
* Flexible homepage layout

For more information about the Keep Calm and E-Comm theme, you can visit <a href="http://etoilewebdesign.com/themes/keep-calm-and-e-comm/">http://etoilewebdesign.com/themes/keep-calm-and-e-comm/</a>.

Instructional video tutorials can be found here: https://www.youtube.com/playlist?list=PLEndQUuhlvSqdDl30dkYU10JeJhdi2jOO

== Installation ==

1. In your WordPress admin panel, go to Appearance > Themes and click the "Add New" button.
2. Type "Keep Calm and E-Comm" in the search box and press Enter.
3. Click the "Activate" button to use the theme right away.
4. Go to Theme Options (in the sidebar in your WordPress admin) to customize the theme to your needs.

OR

1. Upload the "keep-calm-and-e-comm" folder to the "/wp-content/themes/" directory
2. Activate the theme through the Appearance > Themes menu in WordPress
3. Go to Theme Options (in the sidebar in your WordPress admin) to customize the theme to your needs.

Instructional video tutorials can be found here: https://www.youtube.com/playlist?list=PLEndQUuhlvSqdDl30dkYU10JeJhdi2jOO
